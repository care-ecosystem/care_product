### Objective

This SOP explains how to create product knowledge in Care by first selecting or creating the appropriate category, then entering the product details and related attributes. It ensures the product is set up consistently so team members can manage and reference it correctly.

### Key Steps

**1. Open Product Knowledge in Settings** [0:01](https://loom.com/share/e42458784d6c43f698bb962d3952852f?t=1)

![https://loom.com/i/774427307548428094631b3ad3bb97d6?workflows_screenshot=true](../images/external-b5899e50038a.jpg)
- Go to **Settings**.

- Select **Product Knowledge**.

- Confirm you are in the correct module before adding a new product entry.

**2. Select or Create the Correct Category** [0:01](https://loom.com/share/e42458784d6c43f698bb962d3952852f?t=1)

![https://loom.com/i/774427307548428094631b3ad3bb97d6?workflows_screenshot=true](../images/external-b5899e50038a.jpg)
- Check whether the required category already exists.

- If the category is available, select it.

- If not, create the appropriate category(Could be medicines/consumables etc.) first before proceeding.

- In the example, the **Medicine** category is selected.

**3. Enter Product Knowledge Details** [0:19](https://loom.com/share/e42458784d6c43f698bb962d3952852f?t=19)

![https://loom.com/i/ab8f7ef4571543e6a57016ebd6a42242?workflows_screenshot=true](../images/external-8f3e49c46756.jpg)
- Click into **Product Knowledge** entry fields.

- Enter the product name.

- Add the product strength or specification.

- Select the **base unit**.

- Enter any **alternative names** for the product.

- Add any relevant **storage guidelines**.

- Select the **dosage form**.

- Choose the **intended route(s)** of administration.

### Cautionary Notes
- Ensure the category is created or selected correctly before entering product details.

- Double-check spelling, dosage form, and route selections to avoid incorrect product setup.

- Enter storage guidelines accurately, especially for medicines or sensitive products.

### Tips for Efficiency
- Prepare product details in advance to reduce data entry errors.

- Use consistent naming conventions for product names and alternative names.

- Verify all required fields before saving to avoid having to reopen and edit the record.
