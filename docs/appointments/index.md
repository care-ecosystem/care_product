# Appointments & Scheduling

Learn how to manage patient appointments, schedule healthcare services, and handle appointment-related operations.

## Overview

The appointment management system helps schedule patient visits, link billing information, and manage appointment status changes.

## Workflow

1. **Schedule** - Create appointment for patient with desired healthcare service
2. **Configure Billing** - Link charge items for automatic billing
3. **Check-in** - Mark patient as arrived when they reach facility
4. **Manage Changes** - Cancel or reschedule as needed
