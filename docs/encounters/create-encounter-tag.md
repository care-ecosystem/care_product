### **Objective**### This SOP outlines the steps to create a patient encounter tag in the CARE system. ### 

**Key Steps

**### **Step 1: Access Patient Search** [0:00](https://loom.com/share/c471c3bab27940e19670af50d988c437?t=0)
- Navigate to the left navigation bar in the CARE system.

- Click on **Tag Config**.

- Then click on **Add Tag Config** button that shows up on the right hand side.

- Name the tag 

- Select the tag category. 

-  Under the resource option - select the relevant tag type. For example: Patient/Encounter/Account etc. based on the purpose. 

- Select priority - 

100 - Minimun

- 1 - Maximum

-  Add the relevant organisations to which this tag needs to be used

- Click on create tag button
